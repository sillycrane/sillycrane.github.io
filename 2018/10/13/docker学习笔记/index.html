docker学习笔记

发表于 2018-10-13 | 分类于 study

安装docker

添加源并更新

/etc/apt/sources.list添加
deb http://http.debian.net/debian jessie-backports main
运行sudo apt-get update

安装

apt-get install docker.io

查看是否安装成功

docker versions

镜像

dockerhub官网
daocloud挑选镜像并下载

查找镜像

docker search IMAGE

拉取镜像

拉取单个
docker pull IMAGE:TAG
拉取整个镜像
docker pull -a IMAGE

创建镜像

docker commit [-m 注释] [-a 作者] 容器ID IMAGE

删除镜像

docker rmi IMAGE:TAG
docker rmi IMAGEID

基本操作

运行一个容器打印helloworld

docker run IMAGE echo 'helloworld'

启动一个交互式容器名叫cli1

docker run [--name=cl1] -i -t IMAGE /bin/bash

查看容器

docker ps [-a] [-l]

-a显示所有
-l显示最新

查看容器配置

docker inspect 容器名

重启一个容器

docker start [-i] 容器名

删除一个容器

docker rm 容器名

守护式容器

使用ctrl+p和ctrl+q键退出，使其在后台运行
使用以下命令回到守护式容器
docker attach 容器名
docker exec -it 容器 /bin/bash
停止守护式容器
docker stop 容器名发送停止请求
docker kill 容器名直接停止容器

查看容器日志

docker logs [-f] [-t] [--tail] 容器名

-f一直跟踪返回
-t显示时间
–tail显示数量

端口映射

映射所有端口
docker run -P -i -t IMAGE /bin/bash
映射目标端口
docker run -p 80 -i -t IMAGE /bin/bash将80端口随机映射
docker run -p 8080：80 -i -t IMAGE /bin/bash将80端口映射到主机的8080端口

其他

赋权

sudo groupadd docker
sudo gpasswd -a 用户名 docker
sudo service docker restart

IMAGE储存地址

/var/lib/docker

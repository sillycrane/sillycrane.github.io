类和对象

  • 自身类的对象是不可以作为该类的成员的
  • 类不是C语言的结构类型
  • 重载不是面向对象系统包含的要素
  • 面向对象程序设计将数据与对数据操作放在一起，作为相互依存、不可分割的整体来处理
  • 在面向对象方法中，一个对象请求另一个对象为其服务的方式是通过发送消息
  • 在c++语言中，也可以使用struct（结构体）来定义一个类。（结构体默认权限是public）
  • 在建立数组时，要调用构造函数。如果有50个元素，就需要调用50次构造函数。
  • 在建立对象时，编译系统会为每一个对象分配一定的存储空间，以存放其成员。
  • 常对象只能调用常成员函数
  • 常数据成员只能通过参数初始化表初始化，之后不可另外赋值
  • 如果对一个类定义了两个或多个对象，则这些同类的对象之间可以互相赋值
  • 如果在函数中定义静态(static )局部对象，则只在程序第一次调用此函数建立对象时调用构造函数一次。

类内

构造析构

  • /瑞格6203这道题好像出错了/
  • 具有转换函数功能的构造函数，应该是带有一个参数的构造函数
  • 调用内嵌对象的构造函数顺序按照内嵌对象在组合类的定义顺序，而非在初始化列表中出现的顺序
  • 析构函数名与类名相同（wtf那个~算什么。。。），并且无参无返，一个类中只能定义一个析构函数（必须在类体内定义。。。）
  • 析构函数调用与构造函数调用顺序相反
  • 在全局范围中定义的对象(即在所有函数之外定义的对象)，它的构造函数在文件中的所有函数(包括main函数)执行之前调用。
  • 如果定义的是局部自动对象(例如在函数中定义对象)，则在建立对象时调用其构造函数
  • 在C++语言中，如果不自定义类的拷贝构造函数，则每个类都有默认的拷贝构造函数
  • 析构函数不能重载
  • 构造函数不能是虚函数

    静态函数

  • 静态数据成员要在类体外进行初始化
  • 引用静态数据成员时，要在静态数据成员前加<类名* 和作用域运算符
  • 说明静态数据成员时前边要加修饰符static
  • 静态成员函数可以直接访问该类的静态数据和成员函数，然而访问非静态成员必须通过对象名（P157）

    友元

  • 友元并非类的成员
  • C++不允许将构造函数和析构函数声明为友元函数
  • 友元关系声明可以放在类定义中的任何地方

    继承

  • 派生类把基类全部的成员(不包括构造函数和析构函数)接收过来，也就是说是没有选择的，不能选择接收其中一部分成员，而舍弃另一部分成员
  • 多继承情况下,派生类的构造函数中基类构造函数的执行顺序取决于定义派生类时所指定的各基类的顺序。
  • 在保护继承中,对于派生类的访问同于公有继承,而对于派生类的对象的访问同于私有继承
  • 对基类数据成员的初始化必须在派生类构造函数中的参数初始化表处进行

    多态

  • 多态性是指具有不同功能的函数可以用同一个函数名，这样就可以用一个函数名调用不同内容的函数。
  • 静态多态性和动态多态性
  • 编译时多态，运行时多态
  • 重载多态，强制多态，包含多态，参数多态
  • 当类中存在动态内存分配时经常将类的析构函数函数声明成虚函数
  • 基类中说明了虚函数后，派生类中与其对应的函数都为虚函数

    运算符重载

  • 运算符重载不可以改变语法结构
  • 运算符重载函数可以是类的成员函数,也可以是类的友元函数,还可以是既非类的成员函数也不是友元函数的普通函数｡
  • 只能重载C++中已经有的运算符，不能定义新运算符

    纯虚函数

  • 如果纯虚函数没有被重载，则派生类将继承此纯虚函数，即该派生类也是抽象类（抽象类不能定义对象）

others

  • new关键词（默认情况下等同于malloc）不可以重载(与瑞格6268题好像有点冲突,百度了一哈好像是可以重载的。。。)
  • vector<元素类型>数组对象名(数组长度,元素初值);使用前包含vector这个库，如果元素类型是类的话，会调用默认的构造函数就行初始化，只能为所有元素指定相同的初值

  • 背答案系列

  • [],+,(),new 这一组不能全被友元函数重载(未知)
  • ->,+,*,>> 这一组不能全被友元函数重载(->)
  • =,+,-,\ 这一组不能全被友元函数重载(\)
  • <<,>>,*,+ 这一组能全被友元函数重载
  • 下面是重载为非成员函数的运算符函数原型，其中错误的是Fraction& operator+(Fraction&,Fraction)
  • 声明"class B: virtual public A"说明类B为虚基类这句话是错的
  • 用指向基类对象的指针变量指向子类对象是合法的、安全的，不会出现编译上的错误
  • 继承是纵向的，组合是横向的
  • 虚函数不得是静态成员函数
  • 含有纯虚函数的类称为抽象类
  • 纯虚函数不能是友元函数
